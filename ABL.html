ABSOLUTE BEGGINER GUIDE TO LINUX

Disclaimers

Who this guide is aimed for?

This guide is aimed for people that want to start using linux, but don't have idea where to start and what to choose.
This guide is made to be absolute beguinner with handholding, removing any and all advanced choices at it's core and help people to boot on their first system and start learning everything linux from there.
This guide will not delve deep in "why you recommend X or Y" as it will only make people more confuse than they already are.
This guide will not teach you how to install any specific linux distro, but will link to other guides as needed. Also will link to distros and programs official sites for easy of search.

What is the objective of this guide?

To, at first moment, remove the choice paralysis that is so common for people that are switching to linux. Also, to add some tips and tricks for beginners.
It's not meant to be a definitive set in stone way to do things. As soon you get confortable with linux you should explore more

Which are my sources on this guide?

The voices in my head, r/Linux, r/linux4noobs, r/linux_gaming, chans, linuxquestions.org, vivaolinux.com.br and a lot of other forums around the web.

STARTING

Should i dual boot?

It's a good ideia to do so as you can switch between systems on boot until you decide if you want to keep on linux or not. I would still recommend for you to install it alone if you can because bootloaders can be messy to deal with.

What should i do before starting to migrate?

First: Take note of your most used programs and search if they are avaliable on linux or if there is a alternative. You can use some windows-only programs on linux through Wine, but it's kinda complicated for a beginner. If you play games you should take a look at protondb and use it's Library search function to see if the games you play are compatible.
Second: Backup everything important (photos, docs, etc) of your pc to a cloud service or external HDD/Pendrive.
Third: Backup your windows key/serial. Fourth: Be mentaly prepared to something go wrong and keep calm.

Which program should i use to make my bootable pendrive?

Rufus if you want only to try a single distro at time. Ventoy if you want to try many without formating the pendrive. You need at least a 4gb pendrive.

Which distro should i use?

There is a lot of distro out there and all of them do basically the same and the difference for a beguinner is what comes pre-installed and pre-configured. Everything boils down to what is your use case.
If most of your time is playing games: Start with Bazzite.
If most of your time is on your browser or deskwork: Start with OpenSUSE or Mint.
Both openSUSE and Mint have a lot of support from community and the devs, but openSUSE is more focused in being stable and up-to-date while Mint is focused in being absolute user friendly. Both are easy to install and use.

Which desktop environment should i use?

There is a lot of differences on look and feel on each one, but it boils down to are you used to windows or mac flow?
If you like windows try KDE (or Cinnamon if you choose Mint). If you like mac/android try GNOME.
If you have an old machine and/or a HDD use XFCE or MATE

What about window managers?

Those are used by more advanced user that like to work in minimalistic and know how to navigate through the system with keyboard-only. They are very good for coding and text editing work flow and you should check them out when you feel more confident about your linux knowledge.

I choose my distro and my desktop, what should i do now?

Follow your choosen distro guide and install it in your machine. Even if the distro say to use any other program to format your pendrive you still can use Rufus or Ventoy.

AFTER INSTALL

Ok, i installed it and it booted. What should i do now?

Open the terminal and update your system.
On Bazzite: "sudo ujust update"
On openSUSE: "sudo zypper refresh" and then "sudo zypper upgrade"
On Mint: "sudo apt update" and then "sudo apt upgrade"
Those three distros have a program to update it, but it's good to start discovering the terminal.
You mai need proprietary codecs too.
On openSUSE: "sudo zypper in opi" and then "opi codecs"
On Mint: take a look here.

How i get new programs?

Each distro have some kind of "appstore".
On Bazzite: Use Discover.
On OpenSUSE: Use YaST.
On Mint: Use Software Manager.
All distros can use flatpak/flathub and it's a good way to install some apps like discord and spotify.
Installing programs thought the terminal is a easy and viable too. You can google "how i install X on Y" and find how to do so.

Which programs should i install?

Those are some programs and tools that are must have to make your life easier:
Timeshift: It's a backup tool that will save your life if something goes wrong. Configure it to auto backup at least 1 time everyday until you feel secure on your distro.
HW-Probe: Helps diagnose your system to see if any part of it has drive problems.
Inxi: Give more detailed info about your pc. It's good info to give out when you ask for help as people can track your hardware for compability issues. (use "sudo apt-install inxi" on mint or "sudo zypper in inxi" on openSUSE)

Someone said that i should use X instead of Y, should i change?

If you're talking about system programs your should change until you feel confident that you can switch up or you need to.
Terminal: Use the one your distro provide. For a beginner there is zero difference any of them.
File Explorer: Same as Terminal.
Browser: People will push for you to use Firefox, but you can use any of the other that you're already confortable with.
Terminal Text Editor: Don't switch to vim or nvim. Nano is pre-installed in all above distros and it's much more user friendly
