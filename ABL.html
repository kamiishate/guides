ABSOLUTE BEGGINER GUIDE TO LINUX

Disclaimers

Who this guide is aimed for?


This guide is aimed for people that want to start using linux, but don't have idea where to start. This guide is made to be absolute beguinner with handholding, removing any and all advanced stuff at it's core and help people to boot on their first system and start learning everything linux from there. This guide will not delve deep in "why you recommend X or Y" as it will only make people more confuse than they already are.

What is the objective of this guide?


To, at first moment, remove the choice paralysis that is so common for people that are switching to linux. Also, to add some tips and tricks for beginners.
It's not meant to be a definitive set in stone way to do things. As soon you get confortable with linux you should explore more

Which are my sources on this guide?


The voices in my head, r/Linux, r/linux4noobs, r/linux_gaming, chans, linuxquestions.org, vivaolinux.com.br and a lot of other forums around the web.
Those subreddits have their own wiki and a lot of info that you should take a look as soon as you feel confident on your understanding of linux.

STARTING

Which distro should i use?


There is a lot of distro out there and all of them do basically the same and the difference for a beguinner is what comes pre installed. Everything boils down to what is your use case.
If most of your time is playing games: Start with Bazzite
If most of your time is on your browser or deskwork: Start with OpenSUSE or POP_OS!

Which desktop environment should i use?


There is a lot of differences on look and feel on each one, but it boils down to are you used to windows or mac flow?
If you like windows try KDE. If you like mac try GNOME.
If you have a old machine and/or a HDD use XFCE or MATE

What about window manager?


Those are used by more advanced user that like to work in minimalistic and know how to navigate with the terminal and/or with keyboard only.
